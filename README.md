# CRUD de Tareas

Proyecto de gestión de tareas desarrollado con Node.js, Express y SQLite.

## Requisitos

- Tener instalado Node.js

## Instalación

1. Descargar o clonar el proyecto:
git clone https://github.com/rosarioduranrosmery0909-beep/crud-de-tareas.git

2. Entrar a la carpeta del proyecto:
cd crud-de-tareas

3. Instalar dependencias:
npm install

## Ejecución

1. Ejecutar el servidor:
node server.js

2. Abrir en el navegador:
http://localhost:3000/crud.html

## Funcionalidades

- Crear tareas
- Ver tareas
- Eliminar tareas
- Marcar tareas
